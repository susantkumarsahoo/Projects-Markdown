### 2D Visualizations (Chart.js)
- Pie Chart
- Donut Chart
- Polar Area Chart
- Multi-Line Chart
- Stacked Area Chart
- Scatter Plot
- Horizontal Bar Chart
- Stacked Bar Chart

### Hierarchical & Flow (Plotly)
- Sunburst Chart
- Sankey Diagram
- Treemap

### Network Visualization
- Custom Canvas-based Network Graph
- Dynamic edge linking based on distance

### 3D Visualizations (Plotly)
- 3D Scatter Plot
- 3D Surface Plot
- 3D Bubble Chart
- Simulated 3D Bar Chart (via `scatter3d`)
